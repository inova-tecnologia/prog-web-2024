-------------------------------

Tema do site:

Bonsai

-------------------------------

Tecnologias:

Javascript com banco de dados Mysql e SQL no Back-end

Angular no Front-end

-------------------------------

Arquitetura:

Monolítica (MVC)

-------------------------------

Escopo do Projeto:

O objetivo é o desenvolvimento de um site para uma loja física que oferece bonsais para compra online, 
com a opção de retirada no local. O site foi dividido em três seções principais:

1. Informações do Cliente: Nesta seção, o cliente poderá inserir seus dados pessoais, que serão utilizados 
para a confirmação do pedido e comunicação com a loja. 

2. Opções de Bonsai Disponíveis: Esta seção exibe os bonsais disponíveis para compra. Cada bonsai é apresentado 
com uma imagem, descrição, preço, e outras informações relevantes, permitindo que o cliente selecione facilmente 
o produto desejado.

3. Carrinho de Compras: Após selecionar os produtos, o cliente poderá visualizar e revisar os itens adicionados 
ao carrinho de compras, verificar os valores totais e confirmar a compra. Cada uma dessas seções foi pensada para
garantir uma navegação fluida e um processo de compra simplificado, contribuindo para uma experiência positiva do 
usuário. O design do site prioriza a clareza na apresentação dos produtos e a facilidade de uso, com o objetivo de
aumentar as vendas online e a satisfação do cliente. 

------------------------------

O que vai ser feito para melhorar ainda mais a usabilidade e 
interação da página  com os futuros compradores: 

Design Responsivo: Garantirei que o site seja totalmente responsivo,
ou seja, que se adapte bem a diferentes tamanhos de tela, como smartphones, tablets e computadores. 
Isso é crucial para que os usuários possam acessar e interagir com o site de maneira eficiente, independentemente do 
dispositivo que estiverem utilizando.

Interface Intuitiva: Adotarei uma interface limpa e minimalista, com uma navegação intuitiva. Usarei ícones e botões 
bem definidos, com rótulos claros, para que os usuários saibam exatamente o que esperar ao interagir com diferentes elementos da página.

Processo de Compra Simplificado: Minimizarei o número de etapas necessárias para concluir uma compra. Vou oferecer a opção de "compra em um clique"
para clientes cadastrados e facilitar o preenchimento de dados com auto-preenchimento.
